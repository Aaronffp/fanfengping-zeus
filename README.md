## 宙斯简介（fanfengping-zeus）
ZEUS平台的设计初衷是做全栈能效平台，利用业务时间完成。当前使用了 Springboot + Mybatis + Vue + MySQL 实现，开发者可以更好更快捷的进行管理系统的研发。
<br><br><br>

```
// install package
npm i

// run
npm run dev

// build
npm run build
```
